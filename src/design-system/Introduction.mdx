# Timds - This Is My Design System

> timds (pronounced "tim-dis") are represents This Is My Design System, a personal project to build a reusable component library and design system for my projects. The goal is to create a cohesive visual language and set of UI components that can be easily integrated into various applications, ensuring consistency and efficiency in design and development.

A React component library with a built-in design system, powered by
**shadcn/ui (new-york)** and **Tailwind CSS v4**. Tokens are defined once as CSS
variables in `src/styles.css` and consumed everywhere through Tailwind's
`@theme inline` bridge.

The visual language is derived from a dark, modern **analytics dashboard**:
deep indigo surfaces, a vibrant violet brand color, and a cyan support accent.

## How tokens flow


<ol style={{ listStyleType: 'decimal' }}>
  <li><strong><code>:root</code> / <code>.dark</code></strong> in <code>styles.css</code> declare the raw <code>oklch</code> values.</li>
  <li><strong><code>@theme inline</code></strong> maps each <code>--token</code> to a Tailwind color utility (e.g. <code>--color-primary</code> → <code>bg-primary</code>, <code>text-primary</code>).</li>
  <li><strong>Components</strong> reference only the semantic utilities — never raw colors — so re-theming is a single-file change.</li>
</ol>

## Using the library

```tsx
import { Button, Card, CardHeader, CardTitle, Badge } from 'timds'
import 'timds/styles.css'

export function Example() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>Revenue</CardTitle>
        <Badge>+9.8%</Badge>
      </CardHeader>
    </Card>
  )
}
```
